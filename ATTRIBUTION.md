# Attribution

Supra Connect for Home Assistant is an independent project by `its-me-prash`.

The repository originally existed as a fork containing `bimmer_connected`, which provided useful historical context,
fixture examples, and BMW/Supra platform research. That upstream project is not the active codebase of this integration.

The current `custom_components/supraconnect` integration is new Home Assistant/HACS work focused on Toyota Supra Connect
telemetry through user-authorized MQTT/Cardata-style payloads. It does not vendor or depend on the old `bimmer_connected`
Python package.

Historical source reference:

- `bimmerconnected/bimmer_connected`: https://github.com/bimmerconnected/bimmer_connected

Platform references:

- Toyota Supra Connect: https://www.supraconnect.com/
- BMW/Supra CarData and MQTT community work noted in the project README.
